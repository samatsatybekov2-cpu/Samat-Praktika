# Интернет-магазин на PHP

Учебный проект интернет-магазина.

## Возможности

- Каталог товаров
- Корзина
- Избранное
- Личный кабинет
- Админ-панель
- Работа с базой данных

## Технологии

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Структура проекта

Основной код находится в папке `shop`.

Для запуска требуется сервер с поддержкой PHP и база данных MySQL.
